## 1. Project Purpose
This project analyzes Apple (AAPL) stock price from 2023 to 2025 to identify trends, volatility, and investment insights for finance students and new investors.

## 2. Data Source
- Source: Yahoo Finance (simulated based on real market trend)
- Time Range: 2023-01-01 to 2025-01-01
- Access Date: 2026-04-19

## 3. Tools & Methods
- Python: pandas, matplotlib, numpy
- Data generation, data cleaning, descriptive analysis, visualization, daily return calculation

## 4. Key Insights
- AAPL shows a clear upward trend during 2023–2025.
- Positive average daily return.
- Moderate and stable volatility.
